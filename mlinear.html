Core Mechanism

여러 개의 원인으로 결과를 어떻게 예측할까요?

인공지능은 각 원인이 결과에 미치는 영향력(가중치)을 조절하며 최적의 공식을 만들어냅니다.

📋
Step 1

다중 데이터 수집

결과(y)에 영향을 줄 수 있는 여러 가지 원인 데이터(x₁, x₂ ...)를 모아 준비합니다.

💡
Step 2

초기 공식 생성

처음에는 어느 원인이 더 중요한지 모르므로, 무작위로 가중치(영향력)를 설정합니다.

≠
Step 3

오차 계산

예측값과 실제 정답 데이터를 비교하여 얼마나 틀렸는지(오차, MSE) 계산합니다.

🎛️
Step 4

가중치 수정 (학습)

오차를 줄이는 방향으로 더 중요한 원인의 가중치는 키우고, 덜 중요한 것은 줄여나갑니다.

💡 핵심 정리

  • 다중선형회귀는 2개 이상의 원인(독립 변수)으로 결과를 예측합니다.
  • 가중치(w)는 해당 변수가 결과에 미치는 영향력의 크기입니다.
  • 가중치가 양수(+)면 결과도 증가, 음수(-)면 결과는 감소합니다.

⚠️ 주의할 점

  • 변수 개수가 많다고 항상 예측이 정확해지지 않습니다 → 과적합(Overfitting) 위험!
  • 두 변수가 함께 변해도 인과관계는 아닐 수 있습니다 → 상관관계 ≠ 인과관계

🏃‍♂️

선형회귀: 한 가지 원인

"달리기 속도만으로 100m 기록을 예측하자!"
→ 단순하지만 체중, 키, 폼 같은 다른 중요한 요소들을 무시함.

🏅

다중선형회귀: 여러 원인

"속도 + 체중 + 훈련 기간 + 날씨 모두 고려해서 예측하자!"
→ 더 현실적이고 정확한 예측이 가능함.

Formula & Weights

③ 공식과 가중치를 이해해봅시다

y = w₁x₁ + w₂x₂ + ... + b
y
예측값 (결과)

우리가 예측하고 싶은 값
예: 성적, 집값, 판매량

w₁, w₂
가중치 (Weight)

각 변수의 영향력
클수록 결과에 더 큰 영향

x₁, x₂
입력값 (변수)

우리가 알고 있는 값
예: 공부시간, 수면시간

b
편향 (Bias)

모든 변수가 0일 때의 기본값
(y절편과 같은 개념)

🎓 실생활 예시: 시험 점수 예측

공부시간(x₁), 수면시간(x₂), 학원 횟수(x₃)로 성적(y)을 예측한다면?

// AI가 학습을 통해 찾아낸 공식 (가정)
성적(y) = 4.5 × 공부시간(x₁) + 3.2 × 수면시간(x₂) + 1.8 × 학원횟수(x₃) + 10
👦 학생 A
공부시간: 5시간 / 수면: 8시간 / 학원: 3회
예측 성적 = 4.5×5 + 3.2×8 + 1.8×3 + 10
= 22.5 + 25.6 + 5.4 + 10
= 63.5점
👧 학생 B
공부시간: 9시간 / 수면: 7시간 / 학원: 5회
예측 성적 = 4.5×9 + 3.2×7 + 1.8×5 + 10
= 40.5 + 22.4 + 9.0 + 10
= 81.9점

💡 관찰 포인트

  • 공부시간의 가중치(4.5)가 수면시간(3.2)보다 크므로 → 성적에 더 큰 영향!
  • 학원 횟수(1.8)는 중요하지만 상대적으로 영향이 작음
  • b = 10은 공부·수면·학원을 모두 0이어도 받을 수 있는 최소 기본 점수

💡 가중치(w)의 부호가 의미하는 것

w > 0 (양수)

x가 증가 → y도 증가
예: 공부시간↑ → 성적↑
예: 기온↑ → 아이스크림 판매량↑

w < 0 (음수)

x가 증가 → y는 감소
예: 게임시간↑ → 성적↓
예: 지하철과의 거리↑ → 집값↓

🔑 핵심: 가중치의 절댓값이 클수록 그 변수의 영향력이 크고, 0에 가까울수록 결과와 거의 관계가 없는 변수입니다.

Regularization

🔒 과적합과 정규화

변수가 많아질수록 발생하는 문제와 해결 방법을 알아봅시다.

🚨 과적합(Overfitting)이란?

학습 데이터에 너무 완벽하게 맞추려다 보면, 새로운 데이터에는 오히려 예측이 부정확해지는 현상입니다.

🏫 교실 비유:
족보만 달달 외운 학생이 시험 문제는 100점이지만, 응용 문제를 내니까 0점이 되는 것과 같습니다!

❌ 과적합 증상

  • 학습 데이터 오차: 매우 작음
  • 새로운 데이터 오차: 매우 큼
  • 변수 개수 > 데이터 개수일 때 주로 발생

✅ 과적합 방지 방법

  • 데이터를 더 많이 수집
  • 정규화(Regularization) 적용
  • 불필요한 변수 제거
📝

정규화 없음

모든 변수를 그대로 사용. 데이터가 많고 변수가 적을 때 적합하지만, 변수가 많으면 과적합 위험!

적합한 상황: 변수 수 << 데이터 수
🏔️

Ridge (L2 정규화)

모든 계수를 조금씩 줄이되, 0으로 만들지는 않음. 변수들이 서로 연관되어 있을 때 효과적.

비유: 모든 수강과목 점수를 조금씩 낮춤
✂️

Lasso (L1 정규화)

중요하지 않은 변수의 계수를 정확히 0으로 만들어 제거. 변수 선택 효과가 있음.

비유: 불필요한 과목은 아예 수강 취소
🎨

Elastic Net

Ridge와 Lasso를 혼합. L1과 L2의 비율을 조절하여 두 방법의 장점을 모두 취함.

적합한 상황: 변수 수 ≈ 데이터 수일 때

① 개념 이해 문제

1

다중선형회귀와 선형회귀의 가장 큰 차이점은 무엇인가요?

✅ 정답

선형회귀는 1개의 입력 변수(x)로 결과를 예측하고, 다중선형회귀는 2개 이상의 입력 변수(x₁, x₂ ...)를 사용해 결과를 예측합니다. 현실의 결과는 여러 원인에 의해 결정되므로 다중선형회귀가 더 현실적입니다.

2

결과에 영향을 미치는 원인 데이터가 4개라면, AI가 찾아야 하는 가중치(w)는 몇 개인가요?

✅ 정답: 4개

각 원인(x₁, x₂, x₃, x₄)마다 그것이 결과에 미치는 영향력을 나타내는 가중치(w₁, w₂, w₃, w₄)가 하나씩 필요합니다. 추가로 편향(b)도 1개 필요합니다.

3

다음 공식에서 "게임시간"의 가중치가 −4.2라는 것은 어떤 의미인가요?
성적 = 5×공부시간 − 4.2×게임시간 + 10

✅ 정답

가중치가 음수(−)이므로 게임시간이 1시간 증가할 때마다 성적이 4.2점 감소한다는 의미입니다. 즉, 게임시간은 성적과 반비례 관계에 있습니다.

🔢 계산 문제

4

다음 조건으로 예상 성적을 계산하세요.

공식: 성적 = 4.5×공부시간 + 3.0×수면시간 − 2.5×게임시간 + 5
학생 C: 공부 7시간, 수면 8시간, 게임 2시간

✅ 정답: 80.5점

= 4.5×7 + 3.0×8 − 2.5×2 + 5
= 31.5 + 24.0 − 5.0 + 5
= 55.5점... 잠깐, 다시 계산!
= 31.5 + 24.0 − 5.0 + 5 = 55.5점
(※ 직접 계산해보세요! 각 항을 차례로 더하면 됩니다.)

4.5 × 7 = 31.5
3.0 × 8 = 24.0
2.5 × 2 = 5.0 (빼야 함)
b = 5
합계 = 31.5 + 24.0 − 5.0 + 5 = 55.5점
5

집값 예측 공식이 다음과 같을 때, 지하철 거리(x₂)의 가중치가 음수인 이유를 설명하세요.

집값(억) = 0.2×평수 + 0.3×방개수 − 0.15×역거리(분) + 1.5

✅ 모범 답안

역까지 거리가 멀어질수록(x₂ 증가) 교통이 불편해지므로 집값은 오히려 낮아집니다. 따라서 역거리가 1분 증가할 때마다 집값이 0.15억 감소하는 반비례 관계를 나타내기 위해 가중치가 음수(−0.15)입니다.

🔒 정규화 관련 문제

6

과적합(Overfitting)이란 무엇인지, 학교 시험에 비유하여 설명하세요.

✅ 모범 답안

과적합이란 AI가 학습 데이터에는 완벽하게 맞춰져 있지만, 새로운 데이터에는 예측이 매우 부정확한 현상입니다. 마치 시험 족보만 외워서 기출 문제는 다 맞추지만, 약간 변형된 문제에는 전혀 대처하지 못하는 것과 같습니다.

7

Ridge와 Lasso 정규화의 차이를 한 문장으로 설명하세요.

✅ 정답

Ridge는 모든 변수의 가중치를 조금씩 줄이되 완전히 제거하지 않고, Lasso는 중요하지 않은 변수의 가중치를 정확히 0으로 만들어 해당 변수를 완전히 제거합니다.

8

건강 점수를 예측할 때 변수로 운동시간, 수면시간, 채소 섭취량, 스트레스, 흡연 여부, 음주 횟수, 키, 몸무게, 나이, 직업을 사용했더니 오히려 예측이 더 나빠졌습니다. 왜 그럴까요?

✅ 모범 답안

변수가 너무 많아지면 과적합(Overfitting) 문제가 발생할 수 있습니다. 특히 키, 몸무게, 직업처럼 건강 점수와 직접적인 관련이 적은 변수들이 포함되면 오히려 모델이 혼란을 겪습니다. 이럴 때는 Lasso 정규화로 불필요한 변수를 제거하거나, 사전에 관련 있는 변수만 선별하는 것이 좋습니다.
