지도학습

Supervised Learning

정답이 있는 데이터로 학습합니다. 입력과 정답을 함께 주면 AI가 규칙을 찾습니다.

📥 입력 + 정답
→
🤖 AI 학습
→
📤 예측
📌 이 교재의 알고리즘: 선형회귀, 다중선형회귀, 로지스틱회귀, SVM, KNN, 의사결정트리
🔍

비지도학습

Unsupervised Learning

정답 없이 데이터의 패턴이나 구조를 스스로 발견합니다. 비슷한 것끼리 묶어냅니다.

📥 입력만
→
🤖 AI 탐색
→
🗂️ 군집화
📌 이 교재의 알고리즘: K-평균 군집화 (K-Means Clustering)
🎮

강화학습

Reinforcement Learning

보상과 벌칙을 통해 최적의 행동을 스스로 학습합니다. 게임 AI에 많이 사용됩니다.

선형회귀

직선으로 데이터의 경향 찾기

1

직선의 방정식 → y = ax + b

2

오차(MSE) 계산 → 예측값과 실제값 차이

3

경사하강법 → 오차를 줄여 최적화

4

활용 예시 → 키/몸무게, 폭염/음료 판매량

💡 포함 내용: 오차(MSE/RMSE), 경사하강법, 학습률, 퀴즈 10문제

체험하러 가기 →

다중선형회귀

여러 요인을 고려한 예측

1

다변수 방정식 → y = a₁x₁ + a₂x₂ + ... + b

2

여러 특성 → 복합적인 요인 반영

3

활용 예시 → 키, 나이로 몸무게 예측

💡 핵심 개념: 다변수, 복합 요인, 평면/초평면

학습하러 가기 →

로지스틱회귀

Yes/No 분류 문제 해결하기

1

확률 예측 → 0과 1 사이의 값

2

S자 곡선 → 시그모이드 함수

3

활용 예시 → 합격/불합격 예측

💡 핵심 개념: 확률, 이진 분류, 시그모이드

체험하러 가기 →

K-최근접 이웃 (KNN)

가장 가까운 이웃으로 분류하기

1

거리 계산 → K개 이웃 찾기

2

다수결 → 가장 많은 클래스로 분류

3

활용 예시 → 고객 취향 분류

💡 핵심 개념: 거리, K값, 다수결

체험하러 가기 →
✂️

서포트 벡터 머신 (SVM)

최대 마진으로 경계 찾기

1

서포트 벡터 → 경계선에 가장 가까운 점

2

마진 최대화 → 두 클래스 사이 간격 최대

3

활용 예시 → 이미지 분류

💡 핵심 개념: 결정 경계, 마진, 커널

학습하러 가기 →

의사결정트리

질문을 통해 답 찾아가기

1

나무 구조 → 가지치며 분류하기

2

정보 이득 → 가장 좋은 질문 선택

3

활용 예시 → 동물 종 분류

💡 핵심 개념: 트리, 노드, 엔트로피

K-평균 알고리즘

비슷한 데이터끼리 묶기

1

중심점 설정 → K개의 기준점

2

반복 계산 → 중심 이동하며 최적화

3

활용 예시 → 고객 그룹 분석

💡 핵심 개념: 중심점, 반복, 비지도 학습

📚 머신러닝 알고리즘 학습 가이드

📖 회귀 알고리즘

숫자를 예측하는 알고리즘입니다. 키로 몸무게 예측, 폭염일수로 음료 판매량 예측하기 같은 문제를 해결합니다. 선형회귀에는 오차(MSE)와 경사하강법 내용이 포함됩니다.

🎯 분류 알고리즘

범주를 구분하는 알고리즘입니다. 사진 속 동물이 고양이인지 강아지인지, 이메일이 스팸인지 아닌지 판단합니다.

🎨 군집화 알고리즘

정답 없이 비슷한 데이터끼리 묶는 알고리즘입니다. 고객을 비슷한 소비 패턴끼리 그룹으로 나눌 수 있습니다.
